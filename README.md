# Longshore Studios 🦎

Riippumaton pelistudio, joka tekee selaimessa pyöriviä pelejä ja julkaisee
niitä Androidille, iPhonelle ja työpöydälle. Pääkehittäjä: **Hacker_Lisko**.

👉 **Sivusto:** https://longshoreforrest.github.io/studios/

## Pelit

Studion peleihin kuuluu mm. Lisko Racing 🦎, BetterCraft ⛏️, Car Crash
Simulator 🚗, Kalastus 🎣, Viuhtin Hoito 🐶 ja Tallipeli 🐴. Koko lista,
kuvaukset ja latauslinkit löytyvät sivustolta.

## Tyylit

Sivustolla voi vaihtaa kuuden eri teeman välillä (Midnight, Daylight,
Synthwave, Forest, Arcade, Hacker) oikean yläkulman teemavalitsimesta.

## Tekninen

Staattinen sivu (HTML + CSS + vanilla JS), ei build-vaihetta. Avaa `index.html`
selaimessa tai tarjoile millä tahansa staattisella palvelimella.
